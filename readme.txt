说明：
​1.文件结构
├── C2_Server.py        # 服务端主程序（本机）
├── SessionManager.py   # 会话管理模块（本机）
└── C2_Client.py        # 客户端程序（虚拟机）

2.​启动顺序

# 服务端
python C2_Server.py
##########################

##########################
# 客户端（需部署到目标机器）
python C2_Client.py
